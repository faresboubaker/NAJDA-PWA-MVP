# NAJDA_PWA_MVP

Project: NAJDA – Emergency & Social Response Platform (Web PWA) – MVP  
Developer & Innovator: Fares Boubaker, Volunteer Paramedic  

---

## Description
NAJDA is a web-based PWA (Progressive Web App) that instantly connects people in danger with nearby volunteers and responders. This MVP demonstrates the concept, interface, and real-time alert system.

Key Objectives:
- Provide a structured support network for communities
- Encourage volunteerism and community responsibility
- Complement official emergency services
- Promote social innovation and technology for emergency response

---

## Features (MVP)
- Emergency type selection: Fire, Bleeding, Heart Arrest, Traffic Accident, Drowning
- Live GPS map
- Real-time alerts
- Responder profiles
- Confirmation: “Help is on the way”
- Simple, intuitive UI suitable for all users
